Bootstrapping PyMR
---------

1. Clone the repository & cd into it
```
git clone https://github.com/kpurdon/pymr.git
```
2. Install [fabric](http://www.fabfile.org/en/latest/)
```
pip install fabric
```
3. Run the bootstrap
```
fab bootstrap
```
4. Run the tests
```
fab test
```
5. Develop!
